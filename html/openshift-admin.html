Benutzer

Benutzer benötigen Zugriff auf den OpenShift Cluster sowie auf die Projekte für die sie entweder verantwortlich sind oder denen sie zugeteilt wurden. Um Zugriff auf den Cluster zu erhalten muss sein Account im Windows AD Mitglied der entsprechenden Gruppe sein.
Je nach Rolle im Unternehmen benötigt der Mitarbeiter auch Zugriff auf die Projekte, dies wird über die nachfolgenden Kommandos realisiert.
